STEP-BY-STEP SETUP
================================================================================

STEP 1: Get OpenAI API Key
   1. Visit https://platform.openai.com/api-keys
   2. Sign up or log in
   3. Click "Create new secret key"
   4. Copy the key (starts with sk-)
   5. Add $5 billing credit

STEP 2: Create Solana Wallet
   1. Install Phantom: https://phantom.app/
   2. Create new wallet
   3. Switch to "Devnet" in settings
   4. Go to Settings â†’ Security & Privacy â†’ Export Private Key
   5. Copy the private key

STEP 3: Get Free Devnet SOL
   1. Copy your wallet address from Phantom
   2. Visit https://solfaucet.com/
   3. Paste address and select "Devnet"
   4. Request airdrop (you'll get 1-2 SOL)

STEP 4: Deploy to Vercel
   1. Push code to GitHub
   2. Go to https://vercel.com/new
   3. Import your repository
   4. Click "Deploy" (without variables first)

STEP 5: Add Variables to Vercel
   1. Go to your project in Vercel
   2. Click Settings â†’ Environment Variables
   3. Add all 4 required variables above
   4. Make sure to check all 3 environments for each

STEP 6: Redeploy
   1. Go to Deployments tab
   2. Click "..." menu on latest deployment
   3. Click "Redeploy"
   4. Wait for deployment to complete

STEP 7: Update App URL
   1. Copy your Vercel URL (e.g., https://your-app.vercel.app)
   2. Go back to Environment Variables
   3. Update NEXT_PUBLIC_APP_URL with your actual URL
   4. Redeploy again

STEP 8: Test Your App
   1. Visit your Vercel URL
   2. Click "Get Started"
   3. Create a profile
   4. Click "Find Project Matches"
   5. If AI responds, it's working! ðŸŽ‰

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "OpenAI API key not found"
Solution: 
   - Check variable name is exactly: OPENAI_API_KEY
   - Verify key starts with sk-
   - Ensure all 3 environments are checked
   - Redeploy after adding

Problem: "Solana private key not configured"
Solution:
   - Check variable name is exactly: SOLANA_PRIVATE_KEY
   - Verify key is in base58 format (not JSON)
   - Ensure wallet has SOL (use faucet)
   - Redeploy after adding

Problem: "Insufficient funds"
Solution:
   - For devnet: Visit https://solfaucet.com/
   - For mainnet: Transfer SOL to wallet
   - Wait a few seconds and try again

Problem: Build fails
Solution:
   - Check all variable names match exactly
   - Ensure no extra spaces in values
   - Verify all required variables are set
   - Check Vercel build logs for specific error
